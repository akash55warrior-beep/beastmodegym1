## Deployment on GitHub Pages

1. Create a GitHub Repository.
2. Upload all files.
3. Open Repository Settings.
4. Click on Pages.
5. Select:
   - Source: Deploy from Branch
   - Branch: main
   - Folder: /root
6. Save.

GitHub will generate your live website URL.
